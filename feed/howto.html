How to configure your nook to install apps

You'll need to allow your nook to install 3rd party applications to install any of these apps here.

What I've done is to update the settings.db using sqlite3. Here are my commands (in bold) lines broken for readability -- please run all commands on a single line.

$ adb shell   <-- from my PC
# cd /data/data/
        com.android.providers.settings/databases
# sqlite3 settings.db
SQLite version 3.5.9
Enter ".help" for instructions
sqlite> update secure set value=1
        where name='install_non_market_apps';
sqlite> .q
# Ctrl-D
$
Reboot your Nook, and it should now allow 3rd-party installs to occur from Trook.
